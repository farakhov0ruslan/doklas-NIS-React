# ATLAS — Руководство по ручному тестированию

## Запуск

```bash
npm run dev        # http://localhost:5173  (MSW-моки активны)
npx vitest run     # авто-тесты (174 шт.)
npx vite build     # проверка сборки
```

MSW перехватывает все `/api/*` запросы — бэкенд не нужен.
Задержки намеренные: GET /places ~300ms, POST /recommendations ~500ms, POST /route/generate ~800ms.

---

## Флоу 1 — Первый визит (онбординг)

**Начало:** открыть `/` в приватном окне (чистый localStorage)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть `/` | Hero-секция с фото, форма поиска |
| 2 | Нажать **ПОКАЗАТЬ** | Появляется `OnboardingModal` поверх страницы |
| 3 | Нажать **Escape** | Модал закрывается, остаёмся на главной |
| 4 | Снова нажать **ПОКАЗАТЬ** | Модал появляется |
| 5 | Нажать **Начать опрос** | Редирект на `/onboarding`, шаг 1 |
| 6 | Нажать **Далее** без выбора интересов | Кнопка неактивна, переход заблокирован |
| 7 | Выбрать 2–3 интереса | Кнопка **Далее** активируется |
| 8 | Нажать **Enter** (с клавиатуры) | Переход на шаг 2 |
| 9 | Пройти все 5 шагов | Прогресс-бар заполняется |
| 10 | На последнем шаге нажать **Готово** | Редирект на `/places`, персональные рекомендации |

**Проверить дополнительно:**
- Кнопка **← Назад** скрыта на шаге 1, появляется на шаге 2+
- После завершения онбординга повторное нажатие ПОКАЗАТЬ ведёт сразу на `/places`

## Флоу 3 — Просмотр мест и фильтрация

**Начало:** `/places`

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Дождаться загрузки | Спиннер → секции по категориям |
| 2 | Нажать на категорию (например, **🎵 Музыка**) | Фильтрованная сетка карточек, появляется сорт-бар |
| 3 | Нажать **Цена ↑** в сорт-баре | Карточки сортируются от дешёвых к дорогим |
| 4 | Нажать **Цена** повторно | Направление меняется на ↓ |
| 5 | Нажать кнопку воронки (☰) в фильтрах | Открывается `PlaceFilterPanel` снизу |
| 6 | Выбрать минимальный рейтинг **4+** | — |
| 7 | Потащить слайдер цены до ~1000 | Метка «до 1 000 ₽» |
| 8 | Нажать **Применить (2)** | Панель закрывается, счётчик 2 на кнопке воронки |
| 9 | Карточки обновляются | Остаются только места с рейтингом ≥4 и ценой ≤1000 |
| 10 | Снова открыть панель → **Сбросить всё** | Все места снова видны, счётчик исчезает |

**Дополнительно:**
- Нажать **Все** в фильтрах → показываются все секции по категориям
- Прокрутить вниз → кнопка **↑** появляется слева внизу, нажать — прокрутка наверх

## Флоу 6 — Подборка (Корзина)

**Начало:** добавить 3–5 мест через кнопки **+ В маршрут**, затем открыть `/cart`

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть `/cart` | Список мест, итоговая стоимость и время |
| 2 | Перетащить карточку в другую позицию (drag & drop) | Порядок изменяется |
| 3 | Нажать **✕** на одном месте | Место удалено, счётчик уменьшился |
| 4 | Нажать **Очистить** | Появляется `ConfirmModal` |
| 5 | В модале нажать **Отмена** | Список не изменился |
| 6 | Снова **Очистить** → **Подтвердить** | Подборка пуста, EmptyState с кнопкой |
| 7 | Вернуть 3–5 мест и нажать **Построить маршрут** | Спиннер ~800мс → редирект `/planner` |

## Флоу 7 — Планировщик маршрута

**Начало:** `/planner` (должен быть построен маршрут)

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Просмотреть маршрут | Дни, время начала/окончания каждого места |
| 2 | Нажать **✕** на месте | Место удаляется из дня |
| 3 | Перетащить место внутри дня | Порядок в дне изменяется |
| 4 | Перетащить место в другой день | Место перемещается между днями |
| 5 | Нажать **Облегчить** в панели пересчёта | Спиннер ~600мс → маршрут обновляется (меньше мест) |
| 6 | Нажать **Добавить** | Маршрут дополняется |
| 7 | Нажать **Сохранить →** | Редирект `/summary` |

**Прямые переходы (guard):**
- Открыть `/planner` без построенного маршрута → редирект `/cart`
- Открыть `/summary` без маршрута → редирект `/`
